# Prever Start Web 桌面

Prever Start 是我在高中时的代表作, 主要开发于 2009 年.

整理了一下, 有兴趣的同学可以玩玩.

服务器端使用的是 ASP.NET 2.0, 不过 Visual Studio 2013 以网站的形式打开 `src` 目录后 F5 可以直接运行.
